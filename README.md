# 🌤️ Modern Weather App

A sleek and responsive weather application that provides real-time weather data for any city worldwide. Built with a focus on clean UI, performance, and simplicity.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* 💧 Humidity level
* 🌬️ Wind speed information
* 🎨 Modern and minimal UI design
* ⚡ Fast and responsive experience
